Magical ingredients used to craft your templates

HTML 4.01 Strict

You may wonder why HTML 4 not XHTML? Good question. HTML 4 is the most recent, complete, browsers and search engines compliant HTML standard developed by W3C. As an outcome we’ve produced fully semantic, optimized HTML Strict markup. Hand-crafted HTML templates are compatible with upcoming HTML5 standard. Opposite to XHTML, HTML 4.01 Strict is fully supported by all web browsers.

Still not convinced and considering XHTML? Take a look at this article: Beware of XHTML and learn more about HTML 4 advantages.

Requires JavaScript support in browser

Interactive elements and animations found on your website require JavaScript enabled to work correctly. Currently all modern browsers support JavaScript and number of users with JS switched off (for security reasons for example) doesn’t exceed 10%.

UTF-8 character encoding http://en.wikipedia.org/wiki/UTF-8

Your templates have been developed in UTF-8 standard – being most popular and powerful document encoding format out there. Thanks to UTF-8 you can easily insert multi-language content in one template at the same time (it wasn’t possible for ISO before).

UTF-8 encoding is fully supported by all web browsers or major email clients. Besides it is able to represent any character in the Unicode standard, it is also backwards compatible with ASCII
